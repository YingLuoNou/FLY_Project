## 添加wsl代理:
进入wsl用户主目录：`cd ~`  
创建.wslconfig文件：`nano .wslconfig`  
输入并保存:`autoProxy = true`  
重启wsl（在Windows终端）:
```bash
wsl --shutdown
wsl
```
添加代理  
```bash
export http_proxy=http://<windows-host-ip>:<port>
export https_proxy=http://<windows-host-ip>:<port>
```
